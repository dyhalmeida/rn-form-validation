## Tecnologias usadas

| Tecnologia                     | Função                 |
| ------------------------------ | ---------------------- |
| Styled Component               | estilização            |
| Eslint, Prettier, EditorConfig | padronização de código |
| Yup                            | Validação de Schema    |
| Formik                         | lidar com formulários  |
